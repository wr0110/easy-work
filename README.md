## Setup

To run this project, install it locally

- Clone repository
- Navigate to the project directory
- Install and run:

### Using yarn

```bash
yarn install
yarn dev
```

or

### Using npm

```bash
npm install
npm run dev
```

## Tests

- Run tests

```bash
yarn test
```

- Clear jest cache

```bash
yarn test:clear
```
